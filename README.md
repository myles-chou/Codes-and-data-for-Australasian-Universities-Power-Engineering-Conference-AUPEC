The data used is from the file "paper data for An exact optimisation model for electric vehicle charging networks a case study in Hobart"

The code file "Hobart_EV" is used to run the optimisation with default fixed station pricing

The code file "Hoart_EV_variable_price" is used to investigate the effects of variable station pricing strategies
